## Installation

Clone the repository

```bash
git clone https://github.com/vinitparekh17/tshirt-store-nodejs
```

Install dependencies

```bash
yarn install
```

Create `.env` file in the root directory according to the `.env.example` file

## Usage

```bash
yarn start
```
